## Welcome
This API allows you to interact with the Aethos network by submitting Ethereum transaction objects
for validation and processing.

The API is designed to be secure, reliable, and easy to use, providing developers with the tools they
need to integrate seamlessly with the Aethos network.

Aethos Middlewares will be able to be run by anyone in the future, today it is run by Aethos Labs.

## Overview

The Aethos Transaction API enables developers to submit Ethereum transaction objects, verify their compliance with protocol policies, and receive responses indicating the status of their transactions. This document provides a comprehensive guide to the API, including endpoint descriptions, request and response formats, and error handling.

**Base URL**

The base URL for all API requests is:

```bash Aethos Api
      https://api.aethos.io/v1
```
